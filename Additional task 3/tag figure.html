Элементы <figure> и <figcapture>

Элемент <figure>

Элемент <figure> используется в сочетании с <figcaption> и предназначен для разметки диаграмм, иллюстраций, фотографий, примеров кода и других вещей.

Элемент <figcapture>

Элемент figcaption представляет собой заголовок или легенду для figure.

Преимущества применения <figure> перед <img >

  1. Предназначен для разметки диаграмм, иллюстраций, фотографий, примеров кода и других вещей.(Широта применения)
  2. Группирует несколько изображений с единым заголовком

При этом задействовать элемент <figure> не всегда может быть целесообразно. Например, графический баннер не должен верстаться с <figure>. Вместо этого используется элемент <img>.
